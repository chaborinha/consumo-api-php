# Integração de API com PHP

Este é um projeto simples desenvolvido para estudar a integração de uma API com PHP utilizando a biblioteca cURL.

## Funcionalidade

O projeto realiza uma requisição para a API do OpenWeather (https://openweathermap.org/api) para obter a temperatura de uma cidade fornecida pelo usuário. 

## Tecnologias Utilizadas

- PHP
- cURL (para realizar as requisições HTTP)
- API OpenWeather (para obter dados meteorológicos)
